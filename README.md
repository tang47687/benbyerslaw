# Ben Byers Law Scraper

A lightweight web scraper built in Go that extracts the latest legal articles and resources from [BenByersLaw.com](https://benbyerslaw.com/).

## Features

- Scrapes the homepage for recent legal guides and articles.
- Extracts article titles and direct URLs.
- Built using the powerful `gocolly` framework.
- Clean and structured output in the terminal.
